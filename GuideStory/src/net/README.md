# net/ — 네트워크 / 데디케이트 서버 (ADR-001)

3단계 멀티플레이어 풀스택 전환의 핵심.

- **서버 권위 구조**: 클라이언트는 입력 패킷 전송, 서버가 지형·충돌을 판정 후 결과를 브로드캐스트.
- WinSock 기반 패킷 구조체 정의(헤더/타입/페이로드).
- **IOCP 또는 Select** 모델, 멀티스레드 세션 관리.
- 증명 과제: IOCP vs Select 비교, 멀티스레드 서버에서 N명이 한 맵에서 동기화되는지 검증. (→ ADR-001)

> 패킷 검증: 비정상 패킷은 거부/로깅. ([../../../RELIABILITY.md](../../../RELIABILITY.md) §2.2)
